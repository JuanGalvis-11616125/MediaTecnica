¿Qué es Programar?

Programar es es crear programas que exhiban un comportamiento deseado

¿Ventajas de Programar?

  1. Desarrolla habilidades para la solución de problemas
  2. Mayores oportunidades de empleo
  3. Proporciona mayor capacidad de adaptación al estilo de vida actual
  4. Existen diveros lenguajes para aprender y trabajar con el que mas te guste
